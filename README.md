# devops-10-exo-start

## Description

Vanilla JS app used to apply devops principles

-   TDD
-   CI
-   CD
-   Git
-   eXtreme Programing

### Prerequisites

-   [Node JS / NPM](https://nodejs.org/en/)

### Installing

Install dependencies

```
npm i
```

### Testing

After installing dependencies, run following command to test

```
npm test
```
